## Cara Deploy ke jubir.spdindonesia.org SEKARANG

### Setup awal (sekali saja, di laptop Mas + VPS)

1. **Generate SSH key** di laptop (Git Bash / PowerShell):
   ```bash
   ssh-keygen -t ed25519 -C "github-actions-jubir" -f ~/.ssh/jubir_deploy -N ""
   ```

2. **Tambah public key ke VPS** — buka Hostinger panel → VPS → Terminal browser, atau SSH dari laptop:
   ```bash
   ssh root@76.13.196.172
   echo "<PASTE_ISI_jubir_deploy.pub_DI_SINI>" >> /root/.ssh/authorized_keys
   chmod 600 /root/.ssh/authorized_keys
   ```

3. **Save private key sebagai GitHub Secret**:
   - Buka https://github.com/msyafiqwahyudi26/Jubir-Warga/settings/secrets/actions
   - New repository secret: Name = `VPS_SSH_KEY`, Value = paste isi `~/.ssh/jubir_deploy` (mulai `-----BEGIN` sampai `-----END`)

4. **Initial setup di VPS** (sekali saja):
   ```bash
   ssh root@76.13.196.172
   cd /tmp
   git clone https://github.com/msyafiqwahyudi26/Jubir-Warga.git
   bash Jubir-Warga/deploy/setup-vps.sh
   ```

### Setelah setup — workflow auto-deploy

Tinggal `git push origin main` dari laptop. GitHub Actions otomatis:
1. Checkout repo
2. SSH ke VPS pakai key dari secret
3. Rsync repo ke `/var/www/jubirwarga/`
4. Reload nginx
5. Site up-to-date di https://jubir.spdindonesia.org dalam ~1 menit

Bisa juga trigger manual: GitHub repo → Actions tab → Deploy to VPS → Run workflow.
